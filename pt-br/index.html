Introdução

O objetivo deste guia é ajudar aqueles que estejam praticando para competições de programação. Apesar de existirem diversos guias que contém muitas informações acerca de programação competitiva, ainda é difícil encontrar um compilado de tutoriais de diferentes lugares. Para facilitar a busca e o estudo de um tópico específico, apresentamos aqui uma lista contendo links para aulas e explicações para muitos deles. Também listamos recursos extras que podem ajudar durante o treinamento. Você também pode contribuir com este guia, adicionando um link para o seu ou outro material não listado aqui via pull request.

Cada tópico é dividido em subtópicos com tutoriais encontrados pela internet. Todos os materiais listados estão em Inglês ou em Português. Não é necessário seguir a lista em ordem. Além disso, alguns subtópicos de uma categoria podem ser pré-requisitos para subtópicos de outras categorias. Assumimos que o usuário deste guia está familiarizado com conceitos básicos de programação (como Laços de Repetição, Arrays, Recursão, Cálculo de Complexidade, Busca Binária, etc) e sabe como utilizar algumas estruturas, como Set, Map, Stack e Queue do C++. Problemas para os conceitos clássicos e básicos podem ser facilmente encontrados no CSES e no Neps Academy (listados na página de recursos extras). Para os tópicos mais avançados, a maior parte dos tutoriais já contém problemas sugeridos para resolver. Além disso, mais problemas podem ser encontrados em outras plataformas.
